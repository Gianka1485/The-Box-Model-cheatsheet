The Box Model

The Box Model

The box model is a set of properties that define the size parts of an element on a web page.

The box model is made up of 4 elements, the size (height and width), the padding, the border and the margin.

  1. width and height: The height and width of the content.
  2. padding: The amount of space between the border and the content.
  3. border: The thickness and style of the border that surrounds the area of the padding and content.
  4. margin: The amount of space between the border and the outer border of the element.

Size

The size of the content of an HTML element can be defined with the CSS height and width properties.

height and weight can be assigned by different types of measurement values, such as pixels, imperial units, and metric units, etc.

Height = 175 pixels Width = 300 pixels

Padding

The padding generates a space between the border and the size of the content, being very useful to accommodate the content of the box.

The CSS property padding, accepts different measure values, as do the size properties. This CSS property has 4 directions, top, bottom, left, and right, and can have up to 4 median units for the directions.

Padding has 4 ways to assign measures:

  • When it only has one measure value. This applies to all addresses of the box element.
  • When it have two measure values. This assigns the first value to the vertical directions and the second value to the horizontal directions.
  • When it have three measure values assigned. This causes the first value to be assigned to the top address of the box element, the second value to the bottom address, and the third value to the horizontal addresses.
  • When it have 4 assigned measurement values. This makes each value correspond to each of the box element's addresses; top, right, bottom, left, respectively.

Border

The border CSS property is used to generate, as its name suggests, a border to the box element, this border is outside the padding area.

This CSS property has several ways to assign values, the main one consists of three values, size measurement, border style and color.

Margin

The margin is used to determine the size of an area, this being the area outside the border of the box element.

The CSS margin property has an almost identical way to padding when it comes to assigning size values.

Border = 20 pixels, solid, gray
Padding = 25 pixels

CSS Margin Collapse

CSS margin collapse occurs when the top and bottom margins of blocks are combined into a single margin equal to the largest individual block margin.

Margin collapse only occurs with vertical margins, not for horizontal margins.

CSS auto keyword

The value auto can be used with the property margin to horizontally center an element within its container. The margin property will take the width of the element and will split the rest of the space equally between the left and right margins.
